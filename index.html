Dataset and Project Overview

Our data set lists different factors people have and tells us whether or not each person has had a heart attack. We used this data set to create models that will predict how likely it is that someone will have a heart attack in the future.

Integrating Machine Learning

Our project uses Machine Learning models to predict the chances of having a heart attack. Machine learning allows users to submit their data to our computer algorithm, which then creates data-driven suggestions and decisions based only on the input data.

By examining variables such as age, sex, and cholesterol levels, our models predict the chances of a possible heart attack. We also used a confusion matrix that compares the actual target values compared to the values predicted by our model to help refine our accuracy. This helps us understand how well our model is performing, what kind of errors it's making, and an overview of what we need to change to make our model as efficient and accurate as possible.





About our Machine Learning Models

Our models use classification; instead of our models predicting a value through regression they classify data into categories. We used 4 different machine learning models. We utilised a KNN Model, an SVM Model, a Random Forest Model and a Gradient Boosting Model. Below you can find information about each of these models and how we used them.

KNN Model

A KNN model is a supervised learning model that uses the distance between the neighbors around a point to check how similar they are and classify the point. The k is the amount of surrounding points that will be considered.

By using this model, we were able to get an accuracy of 88.5% and a recall value of 0.89 (false positive score of 4).

SVM Model

An SVM model is a model that finds a hyperplane (a divider) in an N-dimensional space.

The hyper plane helps separate and distinctly classify data points. To separate the two classes of data points, there are many possible hyperplanes that could be chosen. We needed to help the computer choose a hyperplane that provides the greatest accuracy when classifying data points, so SVM models were perfect for classification applications like ours.

Conclusion and Final Thoughts

In conclusion, random forests performs the best as it yields the greatest accuracy for predicting the likelihood of having a heart attack. This model uses lots of decision trees that collect samples of the dataset and averages them out to make a final prediction, resulting in a high accuracy

Why is this significant?

By predicting the chances of a possible heart attack ahead of time, we give people the opportunity to take preventative measures such as going to the doctor, exercising more, and cutting out activities that can increase blood pressure.
